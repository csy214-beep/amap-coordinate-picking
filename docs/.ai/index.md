# AI 开发文档索引

欢迎使用高德地图坐标拾取器的 AI 开发文档。本文档按优先级整理，帮助开发者快速了解项目架构和开展开发工作。

## 必须遵守的原则

所有文档、代码中不得使用emoji，如有，请移除

任务完成后必须进行的操作：

- 检查项目是否存在冗余文件、模块、包体、目录，如果有，删除它们
- 将本次更改写入相应的 AI 开发文档中
- 进行日志记录：在日志文件中

## 读取建议

1. **顺序读取**：按照优先级从高到低顺序读取文档
2. **重点关注**：核心文档包含项目的基础信息，是理解项目的关键
3. **模块文档**：按需读取，根据具体任务选择相关模块
4. **配置文档**：遇到配置问题时参考
5. **最佳实践**：作为开发和优化的参考
6. **更新日志**：查看最新变更记录，了解项目现状

## 项目概述

这是一个基于 Vue 3 + TypeScript + Vite 的高德地图坐标拾取工具，主要功能包括：

- 地图绘制（多边形、矩形、圆形）
- 点击获取坐标
- 地址搜索
- 坐标数据导出
